# vt2geojson

## 安装

```
pip install vt2geojson
```

## 使用

Using the CLI:

```
vt2geojson --help
```

```
vt2geojson --layer (poi/building/..) https://tiles.gomap.ae/data/v3/15/21355/14083.pbf
```
